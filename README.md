# Learning Logs

Welcome to **Learning Logs**, a web application that helps users track their learning journey. This platform enables users to create topics and log their progress through detailed entries, making it easier to monitor growth and reflect on what they've learned over time.

## 📜 About
**Learning Logs** is a project built with Django that provides:
- A structured way to organize topics of interest.
- An easy interface for adding entries related to each topic.
- A secure, personalized experience with user authentication.

## 🛠️ Technologies Used
- **Framework:** Django (Python)
- **Frontend:** HTML, CSS
- **Database:** SQLite (default with Django)
- **Hosting:** PythonAnywhere

## 🌟 Features
- **User Authentication:** Secure sign-up, login, and logout functionalities.
- **Topics Management:** Create, view, edit, and delete topics.
- **Entries Management:** Add detailed notes for each topic and view them in chronological order.
- **Responsive Design:** Ensures a seamless experience across devices.

## 🚀 How to Set Up Locally
1. **Clone the Repository:**
   ```bash
   git clone <repository-url>
   cd learning-logs
   ```
2. **Set Up a Virtual Environment:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```
3. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
4. **Run Migrations:**
   ```bash
   python manage.py migrate
   ```
5. **Start the Development Server:**
   ```bash
   python manage.py runserver
   ```
6. **Access the Application:**
   Open your browser and navigate to `http://127.0.0.1:8000/`.

## 📂 Project Structure
